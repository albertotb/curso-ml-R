Modelo de Poisson para regresiÃ³n

El nÃºmero de muertes son valores enteros postivos (contador), luego asumiremos que estas observaciones se distribuyen mediante una Poisson.

Asumiremos que la tasa de muertes cambia de forma proporcional (y homogÃ©nea) en el tiempo.

  1. Describe matemÃ¡ticamente el modelo

Para ello, usa la funciÃ³n stan_glm, especificando la familia de observaciÃ³n adecuada y 5000 muestras para cada una de 4 cadenas.

fit_lin <- stan_glm(deaths ~ year, data = as.data.frame(deaths), family="poisson",
                refresh=1000, iter=5000, chains=4, seed=583829)

  1. Ahora realizaremos algunas predicciones (mediana e intervalos 5-95%), usando 5000 muestras MC.
x_predict <- seq(1993,2023)
N_predict <- length(x_predict)
y_predict <- posterior_predict(fit_lin, newdata=data.frame(year=x_predict), draws = 5000)
mu <- apply(t(y_predict), 1, quantile, c(0.05, 0.5, 0.95)) %>%
  t() %>% data.frame(x = x_predict, .) %>% gather(pct, y, -x)
pfit <- ggplot() +
  geom_point(aes(year, deaths), data = deaths, size = 1) +
  geom_line(aes(x, y, linetype = pct), data = mu, color = 'red') +
  scale_linetype_manual(values = c(2,1,2)) +
  labs(x = 'Year', y = 'Traffic deaths') +
  guides(linetype = F)
